Projeto

O Projeto Contatinhos é uma aplicação web desenvolvida para facilitar o gerenciamento de contatos pessoais e profissionais de forma prática e organizada. Criado com Angular no front-end e Spring Boot no back-end, o sistema permite cadastrar, editar, excluir e filtrar contatos, além de marcar favoritos. Idealizado como parte de um projeto acadêmico, o Contatinhos alia tecnologia e usabilidade para oferecer uma experiência simples e eficiente na administração das suas conexões do dia a dia.
